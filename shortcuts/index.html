How to install

  1. Open this page on your iPhone
  2. Tap the install button above
  3. Shortcuts app opens automatically
  4. Tap Add Shortcut
  5. Say "Hey Siri, GrapplingMap Status" to test

Requires

  1. iOS 16+ with Shortcuts app
  2. Settings → Shortcuts → Allow Untrusted Shortcuts enabled
  3. Internet connection (fetches from GitHub)
